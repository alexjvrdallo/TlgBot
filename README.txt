
=========================
📦 BOT DE TELEGRAM - README
=========================

Este bot hace lo siguiente:
✅ Da la bienvenida a nuevos miembros del grupo con las reglas.
🚫 Filtra groserías y SPAM (como links).
⚠️ Advierte a los usuarios que infringen las reglas.
👮 Notifica a los administradores.
🔇 Silencia automáticamente a usuarios reincidentes por 5 minutos.

---------------------------
📁 ARCHIVOS INCLUIDOS:
---------------------------
- bot_telegram.py  -> Código principal del bot

---------------------------
🚀 CÓMO USAR:
---------------------------

1. Instala Python 3 si aún no lo tienes:
   https://www.python.org/downloads/

2. Instala la librería necesaria en tu terminal o consola:
   pip install python-telegram-bot

3. Reemplaza en `bot_telegram.py`:
   - "AQUÍ_VA_TU_TOKEN" → por el token que te dio @BotFather.
   - Lista ADMIN_IDS → coloca los ID de los admins de tu grupo.

4. Ejecuta el bot:
   python bot_telegram.py

5. Agrega el bot a tu grupo y dale permisos de:
   - Leer mensajes
   - Eliminar mensajes
   - Silenciar usuarios

---------------------------
🛠 SUGERENCIAS:
---------------------------
- Puedes agregar más palabras ofensivas a la lista GROSERIAS.
- Puedes personalizar las reglas y mensajes.
- Si necesitas convertir este .zip a .rar, usa WinRAR.
